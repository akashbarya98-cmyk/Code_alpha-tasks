# 🖼️ Image Gallery Project

A responsive Image Gallery Website built using HTML, CSS, and JavaScript.

## 🚀 Features

- Beautiful image gallery layout
- Image slider functionality
- Next/Previous navigation
- Responsive design (Mobile + Desktop)
- Smooth transitions and animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

Image-gallery-project/
│── index.html
│── style.css
│── script.js


## 📸 How It Works

- Images are displayed in a grid layout.
- Clicking on an image opens the slider.
- Users can navigate using next/previous buttons.
- Fully responsive design for all screen sizes.

## 🎯 Purpose of Project

This project was built to practice:
- DOM manipulation
- Event handling in JavaScript
- Responsive web design
- GitHub project hosting
